# Variable Fonts Demo

An interactive demonstration of variable font axes using Google Sans Flex, created for the **UXDX Oslo - Fonts: Clay to Pixels** presentation.

## 🎨 Live Demo

**[View Demo](https://dewald-els.github.io/uxdx-2026-variable-fonts/)**

## What is This?

This demo showcases the power of variable fonts by allowing you to manipulate multiple font axes in real-time:

- **Weight** (wght): 1-1000
- **Width** (wdth): 25-151
- **Grade** (GRAD): 0-100
- **Roundness** (ROND): 0-100
- **Slant** (slnt): -10 to 10
- **Optical Size** (opsz): 6-144

## Features

✨ **Interactive Sliders**: Adjust each font axis individually to see instant changes

🎬 **Smooth Animation**: Click the "Animate" button to watch the font morph smoothly through random variations

📊 **Live CSS Output**: See the exact `font-variation-settings` CSS code in real-time

🎯 **Presentation Ready**: Clean, focused interface perfect for demonstrations

## Usage

Simply open the page and:
1. Use the sliders to manually adjust font properties
2. Click **Animate** to see smooth, automatic transitions between random font variations
3. Click **Stop** to pause the animation

## Technology

Built with vanilla HTML, CSS, and JavaScript - no build tools required! Uses the local Google Sans Flex variable font file to demonstrate all available axes.
